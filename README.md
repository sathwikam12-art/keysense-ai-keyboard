# KeySense AI Keyboard

KeySense AI is a smart Android keyboard designed to improve cyber safety for children using smartphones.

## Problem
Children sometimes type or search harmful or emotional content such as suicide, violence, or self-harm. Parents usually do not know this until it becomes a serious issue.

## Solution
KeySense AI is a custom Android keyboard that monitors typed text and detects risky words or emotional signals in real time. When such words are detected, the system generates alerts so that parents can be informed.

## Features
- Real-time text monitoring
- Detection of risky words and emotional signals
- Secure alert storage for parents
- Offline privacy-friendly design
- Custom Android keyboard interface

## Technologies Used
- Android Studio
- Kotlin
- Android InputMethodService (IME)
- SharedPreferences
